# AutoKey MVP

Hotkey: Alt+Space
Purpose: fix wrong layout (RU/EN) in text fields by translating the currently selected text.

How it works:
- Select a fragment of text
- Press Alt+Space
- The script reads the selected text from the clipboard
- It converts the text using direct RU/EN key mapping
- It replaces the selection with the corrected text

This is an MVP and intentionally simple.
